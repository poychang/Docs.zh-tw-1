# <a name="cannot-convert-start-value-of-type-39lttype1gt39-limit-value-of-39lttype2gt39-and-step-value-of-39lttype3gt39-to-a-common-type"></a>無法將轉換類型 &#39; 的開始的值&lt;type1&gt;&#39;，限制值的 &#39;&lt;type2&gt;&#39; 和間距值的 &#39;&lt;type3&gt;&#39; 成一般類型
指定的 `For...Next` 控制項值具有不同的類型，因此無效，因為它們無法轉換成一般類型。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  變更一個或多個值的類型，以全部轉換成一般類型。
