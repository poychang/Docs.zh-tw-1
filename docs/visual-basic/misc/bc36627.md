# <a name="evaluation-of-lambda-expressions-is-not-valid-in-the-debugger"></a>Lambda 運算式的評估在偵錯工具中無效
在偵錯工作階段期間，已於 [監看式]  或 [即時運算]  視窗中輸入 Lambda 運算式。 在偵錯視窗中，沒有可用的 Lambda 運算式。  
  
 **錯誤 ID︰** BC36627  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   在程式碼編輯器中測試 Lambda 運算式。
