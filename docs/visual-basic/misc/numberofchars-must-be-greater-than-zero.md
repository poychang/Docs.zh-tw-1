# <a name="numberofchars-must-be-greater-than-zero"></a>NumberOfChars 必須大於零
使用 `PeekChars` 物件的 `TextFieldParser` 方法時，您必須提供大於 `NumberOfChars` 的 `0`值。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請將 `NumberOfChars` 變更為大於 `0`的值。
