# <a name="picture-is-not-valid"></a>圖片無效
您已嘗試將點陣圖、圖示或 Windows 中繼檔以外的圖形格式指派給表單或控制項的 `Picture` 屬性。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
1.  請確定您嘗試載入至 `Picture` 屬性的檔案是 [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)]所支援的有效圖形檔。
