# <a name="file-information-cannot-be-queried-while-open-for-writing"></a>當開啟檔案以進行寫入時，無法查詢檔案資訊
已嘗試取得目前開啟以供寫入之檔案的相關資訊。  
  
## <a name="to-correct-this-error"></a>更正這個錯誤  
  
-   請關閉檔案，然後再次嘗試這項作業。
