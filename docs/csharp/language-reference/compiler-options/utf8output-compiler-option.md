# <a name="-utf8output-c-compiler-options"></a>-utf8output (C# 編譯器選項)
**-utf8output** 選項會使用 UTF-8 編碼來顯示編譯器輸出。  
  
## <a name="syntax"></a>語法  
  
```console  
-utf8output  
```  
  
## <a name="remarks"></a>備註  
 在某些國際組態中，編譯器輸出無法正確地顯示在主控台中。 在這些組態中，使用 **-utf8output**，並將編譯器輸出重新導向至檔案。  
  
 Visual Studio 不提供這個編譯器選項，您亦無法以程式設計方式變更。
